
# 프로젝트 시작하기

SSBI-Query 에서의 프로젝트는 데이터를 분석에 맞게 전처리 하여 분석툴로 내보내는 것까지를 의미합니다.

SSBI - PORTAL 로그인 하여 SSBI-Query 를 클릭하여 Query 를 시작해봅시다.
